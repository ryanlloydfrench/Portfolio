# Portfolio
Codebase for portfolio.

## Requirements
1. Install Node
3. Run ```gem install jekyll bundler```
4. Run ```npm install```
5. Change directory into ```app```
6. Run ```bundle install```

## Local Development
1. Run ```grunt```

## Build
1. Run ```grunt build```

## Deploy
1. Run ```grunt deploy```

## Notes
Deploy location can be modified under the buildcontrol options in the ```Gruntfile.js```.
